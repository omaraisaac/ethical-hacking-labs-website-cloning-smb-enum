# ethical-hacking-labs-website-cloning-smb-enum
Overview
This repository documents two hands-on ethical hacking labs completed as part of a cybersecurity training program. The labs focus on understanding real-world attack techniques from a defensive and educational perspective, emphasizing ethical use, authorization, and security awareness.

The labs included:
1. Website Cloning (Educational Lab)
2. SMB Vulnerability Scanning using Enum4Linux

All activities were performed in a controlled lab environment strictly for learning purposes.
